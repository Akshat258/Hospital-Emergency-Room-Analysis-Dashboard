HOSPITAL ER Operational Efficiency & Patient Satisfaction Dashboard
📌 Project Overview
Developed a comprehensive end-to-end Power BI solution to analyze 9,000+ patient records. The project focuses on optimizing Emergency Room (ER) operations by identifying bottlenecks in wait times, tracking departmental performance, and understanding patient demographics to improve overall satisfaction.

🚀 Key Features

Dynamic KPI Tracking: Real-time monitoring of "Average Wait Time," "Target Missed %," and "Patient Satisfaction Scores."
Heatmap Analysis: A Day/Hour surge map to pinpoint peak ER traffic.
Demographic Profiling: Deep dive into patient age groups and referral patterns.
Automated Data Pipeline: Cleaned and modeled messy healthcare data for high integrity.

🛠️ Tech Stack

Tool: Power BI Desktop
Data Transformation: Power Query (M Language)
Analysis: DAX (Data Analysis Expressions) for advanced measures

📊 Dashboard Insights

Staffing Optimization: Identified that the 20–39 age group represents the largest patient demographic, requiring specific resource allocation.
Bottleneck Identification: Pinpointed specific hours and departments where wait-time targets were consistently missed.
Referral Patterns: Visualized department-wise flow to streamline patient transfers.
